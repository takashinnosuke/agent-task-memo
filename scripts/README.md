# Scripts Overview

このディレクトリには、データ移行やメンテナンス、開発支援を目的としたスクリプトを配置します。

## ベストプラクティス
- スクリプトは Python もしくは Node.js で作成し、依存関係は `requirements.txt` や `package.json` で管理する。
- 冪等性を意識し、同じスクリプトを複数回実行しても問題が起きないようにする。
- 本番データに影響するスクリプトにはドライランモードを用意する。
